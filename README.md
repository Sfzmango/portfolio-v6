# portfolio-v6

Personal portfolio for **Maung Htike** — a fast, modern, single-page site built with
**Astro + Tailwind CSS + React islands**, deployed to GitHub Pages.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Static build deployed to GitHub Pages at `sfzmango.github.io/portfolio-v6`.
